# Para saber mais: criando novos arquivos com o operador (>)

O operador > em Linux é usado para redirecionar a saída de um comando para um arquivo, criando um novo arquivo ou sobrescrevendo um arquivo existente.

Além de redirecionar a saída de um comando para um arquivo, o operador > pode ser usado para criar um arquivo vazio. Isso é útil quando você quer apenas criar o arquivo sem adicionar nenhum texto. O comando funciona assim:

```bash
novo-arquivo.txt
```
Ele criará um arquivo chamado novo-arquivo.txt no diretório atual. Se o arquivo já existir, ele será esvaziado (todo o conteúdo será apagado), mas o arquivo em si permanecerá.
